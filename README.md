# Homebrew Tap for Skuf

## Install

```bash
brew tap skuf-lang/tap https://github.com/skuf-lang/skuf --force-auto-update
brew install skuf
```

## Update

```bash
brew update
brew upgrade skuf
```

## Formula location

The formula lives at `homebrew-tap/Formula/skuf.rb` in the main repo.

When publishing as a standalone tap repo, copy the `Formula/` directory to a `homebrew-tap` GitHub repo.
